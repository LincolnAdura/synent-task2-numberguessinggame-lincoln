## Developer
Lincoln Adura

## Description
An interactive command-line number guessing game built in Python. 
The program generates a random number and challenges the user to 
guess it with helpful hints.

## Features
- 🎲 Generates a random number between 1 and 100
- 💡 Hints system (Too high / Too low)
- 🔢 Counts number of attempts
- 🔄 Play again option after winning
- ✅ Handles invalid inputs gracefully
- 🚫 Rejects numbers outside the 1-100 range

## How to Run
Make sure Python is installed, then run:
```bash
python number_guessing_game.py
```
